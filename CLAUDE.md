# Указания для Claude

## Формат ответов
- Каждый ответ в чате начинай строкой с текущими московскими датой и временем:
  `🕐 ЧЧ:ММ, ДД.ММ.ГГГГ (МСК)`. Брать командой `TZ=Europe/Moscow date "+%H:%M %d.%m.%Y"`
  НЕПОСРЕДСТВЕННО перед ответом, не по памяти (в длинной сессии время из давней
  проверки устаревает на часы и дни).

## Рабочий процесс
- В начале задачи: `git pull`.
- Изменения вносятся в `index.html` / `sw.js` в корне репозитория.
- При КАЖДОМ пуше, меняющем `index.html`/`sw.js`, обновлять константу `APP_VERSION`
  в `index.html` (формат `ГГГГ-ММ-ДД.N` — дата выкладки и номер выкладки за день).
  Она показывается пользователям (форма входа, настройки ребёнка, низ экрана родителя).
  Дату брать командой `date` НЕПОСРЕДСТВЕННО перед коммитом, не по памяти: сессия
  может длиться несколько дней, и дата из давней проверки времени устаревает
  (случилось 2026-07-17: выкатили версию «2026-07-13.1»).
- Автотесты: временный `test.js` в корне (Node), прогнать до зелёного, затем удалить.
- Каждую заметную задачу фиксировать записью в `DEVLOG.md` (обратная хронология, вверху файла).
- Пуш в `main` с описанием коммита на русском; GitHub Pages деплоится автоматически.
- НЕ пушить два раза подряд с интервалом в секунды (например, «фикс» и следом «DEVLOG-хэш»):
  каждый пуш запускает деплой Pages, первый отменяется вторым, а идущие встык деплои
  у GitHub нестабильны (зависают в Queued / падают). Делать оба коммита локально и
  отправлять ОДНИМ `git push`. Хэш коммита в DEVLOG писать во втором локальном коммите
  до пуша, либо ссылаться на заголовок коммита.
- После пуша проверять статус деплоя (GitHub Actions, workflow «pages build and
  deployment»): бывает транзиентный сбой — тогда повторить пустым коммитом.
- Пушить в `main` ОТДЕЛЬНОЙ командой с одним рефом (`git push origin HEAD:main`).
  Пуш сразу двух веток одной командой (`git push origin HEAD:main HEAD:другая`)
  GitHub Pages не подхватывает — деплой не запускается вовсе (проверено 2026-07-12,
  дважды). Служебные ветки при необходимости пушить отдельной командой после
  подтверждения деплоя.
